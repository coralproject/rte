# Introduction

CoralRTE is a lightweight and extensible Rich Text Editor based on React.

## Installation

```bash
npm install @coralproject/coral-rte --save-dev
```

## Demo

```js
<CoralRTE
  features={[
    <Bold key="bold" />,
    <Italic key="italic" />,
    <Blockquote key="blockquote" />,
  ]}
/>
```
