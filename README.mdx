# Introduction

CoralRTE is a lightweight and extensible Rich Text Editor based on React.

## Installation

```bash
npm install @coralproject/rte --save-dev
```

## Demo

      <Bold />,
      <Italic />,
      <Blockquote />,
    ]}
  />
</Playground>
